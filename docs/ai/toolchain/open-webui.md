# Open WebUI 配置笔记

## 部署

```bash
cd openclaw
docker compose up -d
```

## 功能清单
- 多模型切换
- 对话历史导出
- 文档上传 → RAG
- 模型管理（拉取/删除）

## 配置要点
- Web 搜索需 API Key
- 图片生成需对接 Stable Diffusion
- TTS 需配置语音模型
